# GitHub user recent activity summary

Show the recent activity summary of a GitHub user.

## Requirements

* [Rust](https://www.rust-lang.org/tools/install)

## Build

```sh
cargo build --release
```

## Usage

```sh
cargo run --release -- MiLk
```

## Format

```sh
cargo fmt
```
It will use `rustfmt.toml` file for formatting.
